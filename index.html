Vanilla JavaScript

Vanilla JavaScript is plain, standard JavaScript without any libraries or frameworks. It runs directly in browsers, enabling dynamic content, interactivity, and manipulation of HTML and CSS on web pages.

React

React is a JavaScript library for building user interfaces. It uses components to create reusable UI pieces, manages state efficiently, and updates the DOM quickly with a virtual DOM..

Node.js

Node.js is a JavaScript runtime built on Chrome’s V8 engine. It lets developers run JavaScript server-side, build scalable network applications, and handle many connections using non-blocking I/O.

MongoDB

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It allows fast querying, easy scaling, and handles unstructured data, making it ideal for modern web applications.
